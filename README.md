# Flipkart Clone <img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSObJbqsg2wv9eYfRwqi3pWI2-nQk0en1SuOhv9iV8ELA&s=10" width="50px" height="50px"/>

A modern, responsive front-end clone of the Flipkart e-commerce platform. This project recreates the familiar user interface of Flipkart, focusing on clean layouts, smooth navigation, and a pixel-perfect user experience.

## 🚀 Live Demo
Check out the live project here: **[Live Demo Link](https://deven-deshmukh.github.io/Flipkart-Github-page/)** 

---

## ✨ Features
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **Navigation Bar:** Clean header containing the logo, search bar, login button, cart, and seller links.
* **Interactive Carousels:** Smooth sliding banners showcasing top deals and offers.
* **Product Grids:** Beautifully structured sections categorized by electronics, fashion, beauty, and more.
* **Footer:** Comprehensive footer with site links, contact info, and social media icons.

---

## 🛠️ Tech Stack
* **HTML5:** For structured and semantic web layout.
* **CSS3:** For modern styling, flexbox grids, custom animations, and responsiveness.
